# VRL.CHURCH - Uplands 2026

This is the working theme for Q4 release of Valley Real Life's RockRMS theme. Since Rock's core theme is compiled via Sass and the theme properties from the theme.json are inserted into the theme.css on saving the theme in the admin gui, it makes more sense to compartmentalize the VRL theme into it's own link.

## Folder structure and includes

The VRL theme will be written with native uncompiled CSS with a base import and linked on it's own.

## Commenting
I'm using the vscode extension "colorful comments" which use the following syntax.

* "*" Normal, green
* "?" Question, blue
* "!" Warning, red
* "^" Info, highlight yellow
* "todo" WIP, highlight orange
* "~" _____, highlight purple
* "&" _____, highlight pink
